# Fagama

### サイトテーマ
ゲームを通じての交流、発見を深められるコミュニティサイト

### テーマを選んだ理由
特定のゲームについて気軽に語り合えたり、一緒にプレイできるユーザを見つけたい時に利用できるサイトがあればよりゲームが好きになれるのではないかと考え、このテーマにしました。

### ターゲットユーザ
老若男女問わず、ゲーム好きな人かつゲーム仲間としての交流を深めたい方向け

### 主な利用シーン
* 好きなゲームについて様々なユーザと交流できる
* 特定のゲームに於いて一緒にプレイできる人を探せる
* 新たなゲームを開拓できる
